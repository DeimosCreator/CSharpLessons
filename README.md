# Текущий контроль 3.2 – Работа с коллекциями и LINQ

## Описание
Консольное приложение на C#, которое с помощью HashSet хранит уникальные категории покупок, сортирует их по алфавиту с помощью LINQ и сохраняет в файл с последующим чтением, с обработкой возможных ошибок ввода/вывода.
