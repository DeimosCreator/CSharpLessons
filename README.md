# Текущий контроль 2.2 – Введение в ООП: полиморфизм и инкапсуляция

## Описание
Программа реализует иерархию классов книг с использованием инкапсуляции и полиморфизма. Базовый класс Book содержит приватные поля title и author с проверкой на пустые значения, а производные классы FictionBook и NonFictionBook добавляют поля genre и topic соответственно и переопределяют метод DisplayInfo() для вывода расширенной информации. В Main создаётся массив объектов Book, демонстрируется полиморфный вызов DisplayInfo(), а также выполняются тесты с корректными и некорректными данными.
