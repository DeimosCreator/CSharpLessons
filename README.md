# Самостоятельная работа 3.3. – Введение в работу с API

## Описание
Программа реализует работу с внешним API GitHub с использованием HttpClient: выполняется поиск репозиториев по запросу пользователя с поддержкой пагинации. Данные обрабатываются через System.Text.Json, рассчитываются метрики (среднее число звёзд, популярный язык, общее количество). Реализовано кеширование JSON-ответа в файл для повторного использования. Добавлена обработка ошибок (HTTP, JSON, таймаут, лимиты API) и пользовательские исключения. Прогресс загрузки отображается в консоли.
