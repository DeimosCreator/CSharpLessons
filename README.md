# Самостоятельная работа 2.3. – Модификаторы параметров и работа памяти

## Описание
В данной самостоятельной работе исследуется работа с памятью и ссылочными типами в C#. Рассматривается разница между изменением состояния объекта и заменой ссылки на объект с использованием модификаторов ref и out. Также анализируется влияние создания больших объектов на потребление памяти и работа сборщика мусора (GC) при очистке неиспользуемых данных.
