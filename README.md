# Практическая работа 2.3 – Модификаторы параметров и работа памяти

## Описание
Программа представляет библиотеку классов на C# для обработки числовых операций с демонстрацией использования модификаторов параметров in, out и ref. Класс NumberProcessor хранит результат вычислений и реализует методы для вычисления квадрата числа, деления с передачей результата через out и накопления результата с обновлением переданного значения через ref.
