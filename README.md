# Текущий контроль 3.1 – Работа с файлами и потоками

## Описание
Консольное приложение, которое запрашивает у пользователя несколько чисел, сохраняет их в текстовый файл numbers.txt, вычисляет сумму и сохраняет её в бинарный файл sum.bin. Реализована обработка ошибок при работе с файлами, безопасный ввод чисел и использование конструкции using для автоматического освобождения ресурсов.
