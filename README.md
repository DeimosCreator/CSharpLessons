# Практическая работа 2.4 – Основы алгоритмизации. Базовые алгоритмы поиска информации

## Описание
Программа запрашивает у пользователя размер массива (от 1 до 10), затем вводит элементы массива. После этого она находит максимальный элемент, считает количество положительных чисел и выполняет линейный поиск заданного числа, выводя его индекс или сообщение о том, что число не найдено.
