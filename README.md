# Текущий контроль 1.1 – Введение в алгоритмы и базовые структуры данных

## Описание
Консольное приложение EvenNumbersApp на C# запрашивает у пользователя 6 целых чисел, сохраняет их в массив и выводит только чётные числа в формате «Чётное: [число]», используя цикл for и проверку чётности через % 2 == 0.
