# Самостоятельная работа 2.4 – Базовые алгоритмы поиска информации

## Описание
Программа реализует несколько алгоритмов поиска (линейный, бинарный, интерполяционный, экспоненциальный) и сравнивает их производительность на разных типах данных (равномерные и кластерные). Также реализован гибридный алгоритм, который автоматически выбирает наиболее эффективный метод в зависимости от размера массива, сортированности и распределения значений.
