# Текущий контроль 2.1 – Введение в ООП: Наследование

## Описание
Приложение демонстрирует иерархию заказов интернет-магазина. Базовый класс Order содержит OrderId и TotalAmount. Производные классы OnlineOrder и InStoreOrder добавляют уникальные поля (адрес доставки и локацию магазина) и переопределяют метод DisplayInfo() для вывода полной информации о заказе. В Main реализованы позитивный, негативный и граничный тесты.
