# Практическая работа 2.2 – Введение в ООП: полиморфизм и инкапсуляция

## Описание
Разработана библиотека классов на C# для моделирования системы управления библиотекой с использованием принципов инкапсуляции и полиморфизма. Создан базовый класс LibraryItem для медиаресурсов с приватными полями и публичными свойствами, а также методами Borrow(), Return() и виртуальным DisplayInfo(). На его основе реализованы производные классы Book, Magazine и Movie, которые добавляют уникальные свойства (количество страниц, номер выпуска, продолжительность) и переопределяют DisplayInfo() для вывода дополнительной информации.
