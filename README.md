# Текущий контроль 2.3. – Модификаторы параметров и работа памяти

## Описание
Программа демонстрирует разницу между передачей структуры по значению и по ссылке.
Создана структура Point с координатами X и Y. В Main создаётся точка, затем вызываются два метода:
MoveByValue (копия структуры — изменения не сохраняются) и MoveByRef (передача по ссылке — изменения сохраняются).
Выводятся координаты до и после вызовов, что показывает, что только ref действительно изменяет исходный объект.
