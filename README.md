# Практическая работа 3.1 – Работа с файлами и потоками

## Описание
Консольное приложение «Космический архив» имитирует работу архивариуса на космической станции. Пользователь вводит названия пяти миссий и количество полученных за них очков. Программа сохраняет данные миссий в текстовый файл missions.txt, а общее количество очков и среднее значение — в бинарный файл mission_stats.bin. После сохранения программа считывает данные из файлов и выводит их в консоль. В программе реализована проверка корректности ввода и обработка возможных ошибок при работе с файлами.
