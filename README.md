# Практическая работа 1.3 – Анализатор числового массива

## Описание
Программа запрашивает у пользователя массив целых чисел, выполняет его анализ (сумма, минимум, максимум, количество чётных, поиск первого нуля), сортирует данные методом пузырька и выводит результаты в удобном формате.
