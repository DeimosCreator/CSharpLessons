# Самостоятельная работа 3.2. – Работа с коллекциями и LINQ

## Описание
Консольные приложения на C#, демонстрирующие работу с коллекциями (Dictionary, HashSet, List) и применение LINQ для обработки данных. Программы реализуют хранение и управление информацией (оценки студентов, заказы товаров и книги), поддерживают добавление и поиск данных, а также выполняют анализ и вывод статистики с использованием операций LINQ.
